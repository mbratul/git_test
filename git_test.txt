********** All Kinds of git and GitHub command **********

******** First time git command ****
git init 

******** Add a file git command ****
git add .

git commit -m "write a details message for your file "

git remote (remote url link from github)

git push -u origin mastar


******** Update all file in github command ****

git add .

git commit -m "message about the change you did in this file"

git push -u origin master

******** Update a single file in github command ****

git add filename

git commit -m "message about the change you did in this file"

git push -u origin master
